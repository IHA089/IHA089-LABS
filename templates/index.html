Navigating the Digital Realm with Code and Security - Where Programming Insights Meet Cyber Vigilance.

Realistic Learning Environment

IHA089-Labs is a self-contained vulnerable lab ecosystem that provides a safe, offline environment for practicing web application security. Every lab is designed to mimic real-world infrastructures, featuring subdomains, SSL/TLS, local mail services, and even simulated WAFs. This ensures learners experience scenarios similar to modern bug bounty targets, not just theoretical challenges.

Educational & Ethical Use Only

All labs are intentionally vulnerable for personal learning, research, and skill-building. They are strictly for offline use and should never be deployed on the public internet or misused against real systems. IHA089-Labs exists to help security enthusiasts practice responsibly and ethically.

Who Should Use It?

IHA089-Labs is ideal for bug hunters, security researchers, and students. It allows users to practice core web security concepts such as SQL Injection, XSS, CSRF, SSRF, and more through hands-on labs, rather than just learning theory.

Works Offline (localhost setup)

All labs run safely on your own machine for a completely isolated environment.

Subdomain-based Labs (xss.iha089-labs.in, etc.)

Each lab runs on a dedicated subdomain, just like real bug bounty programs, making practice feel authentic.

Integrated Mail Service (mail.iha089-labs.in)

Simulates real email workflows such as registration, password resets, and phishing scenarios.

Local SMTP Server

Local SMTP setup to make it real SMTP server for mail delivery between users and labs.

SSL Certificates Included

Every lab is HTTPS-enabled with SSL/TLS so you can test cookies, HSTS, downgrade attacks, and more.

Open-Source & Community Driven

All labs and configs are open-source. Fork, extend, and contribute new challenges to keep the ecosystem growing.

Python venv Environment Support

Each lab runs inside an isolated Python virtual environment for easy setup, safe dependency management.

Continuous Updates

New vulnerabilities and scenarios are regularly added, ensuring your labs evolve with modern attack surfaces.
